This project was developed in windows 7 with the chrome browser Version 56.0.2924.87 (64-bit)

Open the server with node using the command "node server.js". Then open a browser and type in the address "http://localhost:2406" and the chat room will open.
Please enter a name. If you wish to block users shift double click them. To unblock do the same. To send private messages double click on a user. All other public messages
can be typed in the text input. Press enter to send the message.
